## 1. App Icon (Required)

### Specifications
- **Size**: 1024x1024 pixels
- **Format**: PNG (no transparency) or JPG
- **File**: `app-icon-1024.png`

### AI Prompt
```
Create an iOS app icon for "CukaiPal", a Malaysian tax relief management app.
Design requirements:
- Clean, modern, minimalist style
- Primary color: emerald green (#10b981)
- Icon should feature a stylized "CP" monogram or a combination of:
  - Receipt/document symbol
  - Malaysian Ringgit (RM) currency symbol
  - Calculator or financial symbol
- Background: solid emerald green or subtle gradient (emerald to lighter green)
- Icon should be recognizable at small sizes (60x60px)
- Professional, trustworthy appearance for financial app
- No text besides monogram if used
- Rounded square shape (iOS automatically applies corner radius)
- High contrast, easily readable on both light and dark backgrounds
Style: Modern iOS design language, flat design with subtle shadows
```

### Alternative Minimalist Prompt
```
Design a minimal, geometric app icon for "CukaiPal" tax app.
- Single solid color background: #10b981 (emerald green)
- White or light symbol in center
- Symbol: abstract receipt with checkmark, or calculator icon
- Very clean, flat design
- Perfect for iOS home screen
- Size: 1024x1024px
```

## Tips for Success

1. **Consistency**: Use the same emerald green (#10b981) across all assets
2. **Readability**: Ensure text is legible at thumbnail sizes
3. **Localization**: Consider adding Bahasa Malaysia versions if targeting Malaysian market
4. **A/B Testing**: Create 2-3 variations of screenshots to test which converts better
5. **Seasonal Updates**: Plan to refresh screenshots for tax season (March-April in Malaysia)
6. **Accessibility**: Ensure sufficient color contrast (WCAG AA minimum)

## Malaysian Market Considerations

- Use "RM" for all currency displays
- Consider Bahasa Malaysia text overlays for Malaysian App Store
- Show diversity in lifestyle/marketing shots (reflect Malaysian demographics)
- Reference LHDN (Lembaga Hasil Dalam Negeri) in marketing for credibility
- Tax year in Malaysia follows calendar year (Jan-Dec)
